# メールアドレスの認証
## 機能概要

- **新規登録**: メールアドレスとパスワードでアカウントを作成。
- **ログイン**: 登録済みのアカウントで認証。
- **メール認証**: アカウント作成後、メールアドレスの確認を必須化。
- **入力検証**: 
  - **メールアドレス**: `@` を含む形式を要求。
  - **パスワード**: 6文字以上。

## エラーと対応

### 新規登録に関連するエラー

| **エラー内容**              | **表示メッセージ**                       |
|-----------------------------|------------------------------------------|
| 無効なメールアドレス         | 「メールアドレスの形式が正しくありません。」 |
| 短すぎるパスワード           | 「パスワードは6文字以上で入力してください。」 |

### ログインに関連するエラー

| **エラー内容**              | **表示メッセージ**                       |
|-----------------------------|------------------------------------------|
| 登録されていないメールアドレス | 「このメールアドレスのユーザーが見つかりません。」 |
| 未確認のメールアドレス       | 「メールアドレスが確認されていません。」      |
